### 📂 Key Projects

#### 🟢 [Botify — WhatsApp Bot SaaS](https://github.com/Mukil630/botify)
*A multi-tenant SaaS platform where businesses deploy customized, AI-powered WhatsApp chatbots.*
- Built using **Flask**, **Node.js**, **Groq LLM**, **PostgreSQL**, and deployed on **Railway**.
- Supports automated appointment booking, tiered subscription models, and comprehensive admin analytics dashboard.
- Live in production, handling real user interactions.

#### 🤖 [AI Billing Automation Bot](https://github.com/Mukil630)
*An intelligent Telegram bot designed to automate receipt and invoice processing for retail clients.*
- Built with **Python**, **Groq LLM**, **Telegram Bot API**, and **Google Sheets/Drive API**.
- Uses LLMs to extract structured JSON data from handwritten bill images, auto-generates invoice PDFs, and uploads to Google Drive with automated tax validation.

#### 🖥️ [SGC Billing — Desktop Invoice App](https://github.com/Mukil630)
*A desktop-based billing and invoice generation application.*
- Built with **Electron**, **React.js**, **Puppeteer**, and **Google Drive API** (with OAuth 2.0).
- Enables swift invoice creation, automated headless PDF generation, and secure, direct cloud synchronization.

#### 🧠 [NEXUS v3 — AI Desktop Assistant](https://github.com/Mukil630)
*An Electron-based AI desktop companion featuring automated productivity tracking and smart alert systems.*
- Filters and categorizes system notifications with smart Telegram alerts.
- Automatically logs developer activity and presents insights on a sleek, responsive dashboard.
