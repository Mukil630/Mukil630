### 🚀 Highlighted Projects

#### 🌐 [Zora — The Autonomous Agent Grid](https://github.com/Mukil630/Zora)
*A visual, node-based workflow orchestration engine built to configure, run, and scale autonomous AI agents.*
- **Tech Stack:** React 18, TypeScript, Vite, Node.js Express, Prisma ORM, PostgreSQL, Zustand, Stripe, Docker.
- **Key Work:** Drag-and-drop workflow builder, step-by-step runtime visual tracking, integrated AI error debugger, Stripe billing, and multi-LLM (Claude/OpenAI) node processors.

#### 💬 [Botify — WhatsApp SaaS Platform](https://github.com/Mukil630/Botify)
*A multi-tenant SaaS application allowing business owners to deploy customized, AI-driven WhatsApp support agents.*
- **Tech Stack:** Flask, Node.js, `whatsapp-web.js` (Puppeteer session runner), Groq LLM, PostgreSQL, Railway.
- **Key Work:** Created dynamic QR code scan login to link personal WhatsApp instances without Meta API hurdles. Supports appointment scheduling, billing plans, and a comprehensive admin metrics board.

#### 🛍️ [AURA — Telegram-Managed Storefront](https://github.com/Mukil630/clothing-store)
*An ultra-modern, glassmorphic streetwear storefront backed by a real-time inventory management Telegram Bot.*
- **Tech Stack:** Vite, React, Node.js Express, Telegram Bot API.
- **Key Work:** Integrated custom bot commands allowing store owners to upload new catalog photos, edit stock descriptions, and receive immediate alerts for customer checkouts directly on Telegram.

#### 🧠 [NEXUS v3 — AI Desktop Companion](https://github.com/Mukil630/nexus-desktop)
*An Electron-based desktop application monitoring user productivity with automated Telegram warning alerts.*
- **Tech Stack:** Electron, SQLite3, Telegram Bot API, Node.js main process.
- **Key Work:** Implements OS-level notification classification filters, local logging schema with SQLite3, and mobile text warnings via chat bots.
